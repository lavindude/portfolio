1. https://trends-covid-map.herokuapp.com

The way this application works is there is a python script that runs at 6:30am PST everyday to update the database. This script gathers data from an API and inputs that data into a database. Next, a different script uses linear regression on each state to find the slope of a line of best fit for the last 11 days. The color of the state is green if the slope is negative, yellow if moderate, and red if positive. I set these scripts to run automatically so I do not have to manually do it everyday. That is how I managed the server side. I used HTML and CSS to display my work on a website. I used PHP to access data from the database and color each state depending on the data.

Languages/Technolgies used: HTML, CSS, Python, PHP, SQL

GitHub Repository: https://github.com/lavindude/USCOVID

3. Cannon Ball!

I made a simple game, where cubes fall from the top of the screen and the player has to shoot a cannon (from the bottom right corner) to destroy the cubes. If the cube touches the ground, the game is over. Only one cannon can be present on the screen at any given time so the user cannot spam click cannons. Objects that go out of the view are destroyed to improve performance and cleanliness.

Languages/Technologies used: Unity, C#, Visual Studio

GitHub Repository: https://github.com/lavindude/cannonball

**Note: To see all of the C# scripts I wrote, open up the GitHub repository and go to Cannon Ball/Assets/Scripts.

4. Calendar Discord Bot

This is a discord bot that can keep track of events or deadlines in a discord server. For example, typing the command !addEvent 1.2 HW to AP Calc 9/4/2020 will parse the string and create a new row in a database table with the event type, date, and subject (which is the channel the event was added in).

Languages/Technologies used: Python, Heroku, PostgreSQL

GitHub Repository: https://github.com/lavindude/Calendar-Discord-Bot

5. Server Management Discord Bot

This bot can do basic server management functions. It can delete spam by using an algorithm that checks for the interval between messages from a particular user. If the spam algorithm detects spam twice consecutively by the same user, the user gets muted and send a message to the server admin. This way a real human can check if this was indeed spam or if a user typed really fast. The bot also greets new members of the server when they join. This was the first big programming project I completed outside of school, so if you look in the Main.java file, you may notice that all of my other projects were written in a more modular and organized style than this one.

Languages/Technologies used: Java, Heroku

GitHub Repository: https://github.com/lavindude/ManagementBot
